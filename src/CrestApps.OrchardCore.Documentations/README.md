# CrestApps.OrchardCore Documentation

Documentation site for the [CrestApps.OrchardCore](https://github.com/CrestApps/CrestApps.OrchardCore) repository, built with [Docusaurus 3.9](https://docusaurus.io/).

**Live site:** [crestapps.crestapps.com](https://crestapps.crestapps.com)

## Local Development

```bash
cd src/CrestApps.OrchardCore.Documentations
npm install
npm start
```

## Build

```bash
npm run build
```

This site contains the Orchard Core-specific documentation. The shared framework documentation now lives in the separate `CrestApps.Core` repository at <https://core.crestapps.com>.
