# CrestApps OrchardCore Modules

A comprehensive collection of OrchardCore CMS modules built on the [CrestApps AI Framework](../Framework/README.md). These modules provide AI-powered features, omnichannel communication, enhanced user management, and more — fully integrated with OrchardCore's modular architecture.

## Overview

The OrchardCore modules leverage the CrestApps AI Framework to deliver:

- **AI Chat** — Admin UI and frontend widgets for AI conversations
- **AI Profiles & Connections** — Manage AI providers and model configurations
- **MCP Server** — Expose OrchardCore content as MCP resources
- **Omnichannel** — Unified communication (SMS, Event Grid)
- **User & Role Management** — Enhanced user features and permissions
- **Content Access Control** — Fine-grained content permissions

## Getting Started

```xml
<PackageReference Include="CrestApps.OrchardCore.Cms.Core.Targets" Version="*" />
```

Enable the desired features in **Configuration → Features** in the OrchardCore admin dashboard.
