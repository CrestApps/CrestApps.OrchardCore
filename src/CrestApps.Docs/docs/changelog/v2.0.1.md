# Version 2.0.1

## Repository boundary cleanup

- Removed local `src\Framework` project dependencies from the Orchard Core solution and replaced them with published `CrestApps.Core.*` NuGet package references.
- Flattened Orchard Core projects from `src\OrchardCore\...` into `src\...` so the repository structure now reflects the Orchard-specific code that remains here.
- Renamed the Aspire orchestration host from `CrestApps.Core.Aspire.AppHost` to `CrestApps.Aspire.AppHost` to avoid confusion with the standalone CrestApps.Core repository.
- Removed obsolete `CrestApps.Core.Mvc.Samples.*` startup projects from this solution.
- Refocused the Orchard Core documentation site on Orchard modules, startup apps, and CMS integration guidance. Framework-specific docs now live in the CrestApps.Core repository.
